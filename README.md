# Cartly

A full stack MERN e-commerce project built with **React + Vite** on the frontend and **Node.js + Express + MongoDB** on the backend.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Vite

### Backend
- Node.js
- Express
- MongoDB


---

## 📁 Project Structure

cartly/
